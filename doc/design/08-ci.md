# CI / Release Workflow

Two GitHub Actions workflows: CI runs on every push/PR, and Publish runs when a GitHub release is created.

## CI Workflow

```yaml
# .github/workflows/build.yml
name: CI

on:
  push:
    branches: [master]
  pull_request:
    branches: [master]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm
      - run: npm ci
      - run: npm run build
      - run: npm test -- --coverage
```

## Publish Workflow

Publishing to npm is triggered by creating a GitHub release. Uses OIDC trusted publishing -- no npm token stored in secrets.

```yaml
# .github/workflows/publish.yml
name: Publish to npm

on:
  release:
    types: [published]

jobs:
  publish:
    runs-on: ubuntu-latest
    permissions:
      contents: read
      id-token: write
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          registry-url: 'https://registry.npmjs.org'
      - run: npm ci
      - run: npm run build
      - run: npm test
      - run: npx npm@11.12.1 publish --provenance --access public
```

Key details:
- **OIDC authentication:** `id-token: write` permission allows npm to verify the publish came from this specific GitHub repo and workflow. No npm token needed.
- **Provenance:** `--provenance` flag generates a signed attestation linking the published package to its source commit.
- **Pinned npm:** `npx npm@11.12.1` because Node 20 ships an older npm that lacks full OIDC support.

## How to Release

1. Update `version` in `package.json`
2. Commit and push to `master`
3. Create a GitHub release with a `v`-prefixed tag (e.g., `v2.0.0`)
4. The publish workflow runs automatically

## One-Time Setup

Before the first publish, configure OIDC trust on npm:
1. Create the `@de-otio` npm organization (if not done)
2. Create the `@de-otio/epimethian-mcp` package on npm
3. In the package settings, link the `de-otio/epimethian-mcp` GitHub repository as a trusted publisher
